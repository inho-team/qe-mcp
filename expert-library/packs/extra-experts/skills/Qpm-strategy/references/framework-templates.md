# Strategy Framework Templates

## Framework 1: Lean Canvas

One-page business model. Best for: early-stage validation, startup pitches.

### Template

```markdown
# Lean Canvas: [Product Name]

## Problem (Top 3)
1. [Problem 1]
2. [Problem 2]
3. [Problem 3]
- Existing alternatives: [how users solve this today]

## Customer Segments
- Early adopters: [who will use this first?]
- Broader market: [who follows?]

## Unique Value Proposition
- Single clear message: [why different AND worth attention]
- High-level concept: [X for Y analogy]

## Solution (Top 3 features)
1. [Feature solving Problem 1]
2. [Feature solving Problem 2]
3. [Feature solving Problem 3]

## Channels
- [How you reach customers: SEO, ads, partnerships, direct sales...]

## Revenue Streams
- [Pricing model: subscription, usage, transaction, license...]
- [Revenue estimate per unit]

## Cost Structure
- Fixed: [salaries, hosting, tools...]
- Variable: [CAC, support, infrastructure scaling...]

## Key Metrics
- [Activation rate, retention, revenue per user, LTV, CAC...]

## Unfair Advantage
- [What cannot be easily copied: network effects, IP, expertise, data...]
```

### Guiding Questions
- What is the #1 risk to this business? (Start there)
- Can you describe UVP in one tweet?
- Who are early adopters vs mainstream users?
- What is the path from $0 to first revenue?

## Framework 2: SWOT Analysis

Strategic position assessment. Best for: competitive positioning, strategic planning.

### Template

```markdown
# SWOT Analysis: [Subject]

| | Helpful | Harmful |
|---|---------|---------|
| **Internal** | **Strengths** | **Weaknesses** |
| | - [S1] | - [W1] |
| | - [S2] | - [W2] |
| **External** | **Opportunities** | **Threats** |
| | - [O1] | - [T1] |
| | - [O2] | - [T2] |

## Strategic Actions
- **SO (Strengths + Opportunities):** [leverage strengths to capture opportunities]
- **WO (Weaknesses + Opportunities):** [improve weaknesses to unlock opportunities]
- **ST (Strengths + Threats):** [use strengths to defend against threats]
- **WT (Weaknesses + Threats):** [mitigate weaknesses to reduce threat exposure]
```

### Guiding Questions
- **Strengths:** What do we do better than competitors? What unique resources do we have?
- **Weaknesses:** Where do we lack resources or capability? What do customers complain about?
- **Opportunities:** What market trends favor us? What unmet needs exist?
- **Threats:** What are competitors doing? What regulations or shifts could hurt us?

## Framework 3: PESTLE Analysis

Macro-environmental scanning. Best for: market entry, regulatory assessment, long-term planning.

### Template

```markdown
# PESTLE Analysis: [Market/Product]

| Factor | Key Findings | Impact (H/M/L) | Timeframe |
|--------|-------------|-----------------|-----------|
| **Political** | [government policy, trade, stability...] | | |
| **Economic** | [growth rate, inflation, exchange rates, disposable income...] | | |
| **Social** | [demographics, culture, lifestyle trends, education...] | | |
| **Technological** | [innovation pace, automation, R&D, digital adoption...] | | |
| **Legal** | [regulations, compliance, IP, employment law...] | | |
| **Environmental** | [sustainability, climate, waste, energy costs...] | | |

## Implications for Strategy
- [Top 3 factors that most affect our product/business]
- [Required adaptations or opportunities to exploit]
```

### Guiding Questions
- Which factor poses the highest risk in the next 12 months?
- Which factor creates the biggest untapped opportunity?
- Are any factors changing rapidly enough to disrupt our current model?

## Framework 4: Porter's Five Forces

Industry attractiveness and competitive dynamics. Best for: market entry decisions, competitive strategy.

### Template

```markdown
# Porter's Five Forces: [Industry]

## 1. Threat of New Entrants: [High/Medium/Low]
- Barriers to entry: [capital, regulation, brand, tech, network effects]
- Likelihood of new competitors: [assessment]

## 2. Bargaining Power of Suppliers: [High/Medium/Low]
- Number of suppliers: [few/many]
- Switching cost: [high/low]
- Supplier concentration: [assessment]

## 3. Bargaining Power of Buyers: [High/Medium/Low]
- Number of buyers: [few/many]
- Price sensitivity: [high/low]
- Switching cost for buyers: [high/low]

## 4. Threat of Substitutes: [High/Medium/Low]
- Available substitutes: [list]
- Switching cost to substitutes: [high/low]
- Price-performance of substitutes: [assessment]

## 5. Competitive Rivalry: [High/Medium/Low]
- Number of competitors: [count]
- Industry growth rate: [high/low]
- Product differentiation: [high/low]
- Exit barriers: [high/low]

## Overall Industry Attractiveness: [Assessment]
- Strongest force: [which force constrains profitability most]
- Strategic response: [how to position against dominant force]
```

### Guiding Questions
- Which force most constrains profitability in this industry?
- Where can we build structural advantages?
- Is the industry consolidating or fragmenting?

## Framework 5: Product Vision & Value Proposition

Strategic direction and customer value alignment. Best for: product launch, repositioning, team alignment.

### Vision Statement Template

```
For [target customer]
who [statement of need or opportunity],
[product name] is a [product category]
that [key benefit / reason to buy].
Unlike [primary competitive alternative],
our product [primary differentiation].
```

### Value Proposition Canvas

```markdown
# Value Proposition Canvas: [Product]

## Customer Profile
| Jobs-to-be-Done | Pains | Gains |
|-----------------|-------|-------|
| [Functional job 1] | [Pain 1: frustration, risk, obstacle] | [Gain 1: desired outcome] |
| [Social job 1] | [Pain 2] | [Gain 2] |
| [Emotional job 1] | [Pain 3] | [Gain 3] |

## Value Map
| Pain Relievers | Gain Creators | Products & Services |
|---------------|---------------|---------------------|
| [How we reduce Pain 1] | [How we create Gain 1] | [Feature/service 1] |
| [How we reduce Pain 2] | [How we create Gain 2] | [Feature/service 2] |

## Fit Assessment
- **Problem-Solution Fit:** [Do our relievers/creators address the most important jobs/pains/gains?]
- **Product-Market Fit signals:** [What evidence do we have?]
```

### Guiding Questions
- What is the single most important job the customer is trying to get done?
- Which pain is severe enough that customers will pay to eliminate it?
- Can we state our differentiation in one sentence without jargon?

## Framework 6: Business Model & Pricing Strategy

Revenue model design and pricing. Best for: monetization planning, pricing optimization.

### Pricing Frameworks

| Framework | Description | Best For |
|-----------|-------------|----------|
| **Cost-Plus** | Cost + target margin | Commodity products, manufacturing |
| **Value-Based** | Price = perceived customer value | Differentiated products, SaaS |
| **Competitive** | Price relative to competitors | Crowded markets, price-sensitive segments |
| **Freemium** | Free tier + paid upgrades | Growth-stage, network-effect products |

### Monetization Strategy Template

```markdown
# Pricing Strategy: [Product]

## Pricing Model
- Type: [subscription / usage-based / transaction / license / marketplace]
- Tiers: [free / starter / pro / enterprise]

## Tier Design
| Tier | Price | Target Segment | Key Features | Limit |
|------|-------|---------------|--------------|-------|
| Free | $0 | [who] | [features] | [usage cap] |
| Starter | $X/mo | [who] | [features] | [usage cap] |
| Pro | $Y/mo | [who] | [features] | [usage cap] |
| Enterprise | Custom | [who] | [features] | Unlimited |

## Pricing Rationale
- Value metric: [what unit drives pricing -- seats, API calls, storage, revenue...]
- Willingness-to-pay evidence: [surveys, competitor pricing, pilot data]
- Expansion path: [how ARPU grows over time]

## Key Metrics
- ARPU: [target]
- LTV:CAC ratio: [target, minimum 3:1]
- Gross margin: [target]
- Net revenue retention: [target]
```

### Guiding Questions
- What is the natural value metric customers already think in?
- What triggers upgrade from free to paid?
- Is pricing transparent or sales-negotiated? Why?
- What is the competitive price anchor?
