## How to Reference

워크플로우 스킬이나 에이전트에서 구현 작업 시 해당 기술 스택의 전문가 스킬을 참조:

```
## Expert Reference
구현 시 프로젝트 기술 스택에 맞는 coding-experts 스킬을 참조한다:
→ See skills/coding-experts/CATALOG.md
```
